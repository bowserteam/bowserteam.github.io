Perdu dans la machine

Vous êtes les premiers humains dont les esprits ont été transféré (avec succès...) dans la machine ! Problème : vous êtes perdus.

Après quelques mili-secondes de réflexions (une éternité pour vous désormais) vous arrivez à identifier votre position actuelle : un périphérique USB ! La souris sans doute...

Votre seul espoir de vous en sortir est de rejoindre l'Unité Centrale de Calcul, au coeur de la carte mère. Peut-être ensuite pourrez-vous rejoindre la carte réseau puis la passerelle pour vous plonger enfin dans l'infinité de l'internet du Web !

Ligne éditoriale

Ce parcours vous invite à découvrir différents composants informatique présentés avec un ton décalé dans une ambiance mystérieuse. Le style d'écriture utilisera des phrases courtes pour créer un sentiment d'urgence et d'action.
